PicGo + GitHub + CDN

设置

  • 设定仓库名：TAO1st/pics
    • 设定分支名：master
  • 视实际情况分支名来改变
    • 设定Token
  • GitHub - Settings - Developer settings - Personal access tokens - Generate new token

  •   生成token时要注意选中**repo选项**。

  •     现在的token: `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`

  •     - 存储路径：仓库中的路径，默认为空即可

  •     - 设定自定义域名（CDN）：注意修改**用户名/仓库名@分支名**

  •       `https://cdn.jsdelivr.net/gh/TAO1st/pics@master`
